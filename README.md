# Async_API_sprint_1_team

# Запуск приложения

1. Активировать venv
2. Установить зависимости
```bash
pip install --upgrade pip && pip install -r requirements.txt
```
3. Запуск приложения

```bash
cd src && fastapi dev main.py
```
